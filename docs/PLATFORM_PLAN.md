## Env contract: generated files (no mutation)

```text
services.yaml  →  render-env.sh  →  <app>/.env.platform.generated
```

**Per app repo:**

```text
findoc-ai/backend/
  .env                      # local overrides, never written by platform
  .env.platform.generated   # gitignored; regenerated by make render-env
```

**Load order (later overrides earlier):**

```python
load_dotenv(".env")
load_dotenv(".env.platform.generated")
```

**Suite / hub:** before sibling `make up`, run `make render-env` in each repo. **Do not** implement `sync-platform-env.sh`.

**Doctor:** compare rendered output to on-disk `.env.platform.generated` (not private `.env`).

## Postgres: one database, many schemas

| Schema | App |
|--------|-----|
| `knowledge_studio` | praam-knowledge-studio |
| `findoc` | findoc-ai |
| `prism` | prism |
| `askhr` | askHR |
| `plan_copilot` | production-plan-copilot |
| `pulse` | praam-pulse |

**Platform init** (`infra/postgres/init/01-schemas.sql`): schemas + grants + `CREATE EXTENSION vector` only — never app tables.

**App migrations:** Alembic inside each schema. See [SCHEMA_MIGRATIONS.md](SCHEMA_MIGRATIONS.md) (created during implementation).

---

## Connectivity

- **Container → platform:** Docker DNS on `praam-network` — `postgres`, `redis`, `litellm`
- **Host-run apps** (findoc, plan-copilot): `127.0.0.1:15430`, `127.0.0.1:16380`, `127.0.0.1:3100`
- **`PRAAM_USE_PLATFORM=0`:** per-repo local Postgres/Redis (escape hatch; must stay in CI)

## Versioned platform API

Apps pin:

```makefile
PRAAM_PLATFORM_API ?= v1
-include $(PRAAM_PLATFORM_ROOT)/make/$(PRAAM_PLATFORM_API)/platform.mk
```

**v1 stable targets:** `platform-ensure`, `platform-wait`, `platform-doctor`, `render-env`, `verify-schema`

## LiteLLM (Phase 1)

- Secrets only in `~/.praam/secrets.env`
- Apps use model **aliases** (`fast`, `reasoning`, `embedding`) via gateway — not raw provider IDs in app repos

## Platform SDK

```text
lib/praam_platform/           # shared registry + PlatformClient
sdk/python/                   # uv sync --all-extras (see root pyproject.toml)
sdk/typescript/               # npm run build in sdk/typescript
```

Apps call `PlatformClient(app).load()` at startup — see [CONFIG_API.md](CONFIG_API.md).

Phase 1 also supports legacy `render-env.sh` for apps not yet migrated.
